SA:MP GDK 4 For Windows

This webpage has been created by unofficial build of SA:MP GDK 4 (https://github.com/Zeex/sampgdk) made and dedicated for Windows system only.

Download .zip Download .tar.gz View on GitHub

How to do working with Visual Studio:

  1. Download archive file from above.
  2. Unpack the archive to any folder on your hard disk.
  3. Set PATH to this folder.
  4. In Visual Studio add in Properties -> VC++ Directories "PATH-TO-FOLDER-WITH-EXTRACTED-ARCHIVE" (without the quotes), exactly in:

    Executable Directories - write
    PATH-TO-FOLDER-WITH-EXTRACTED-ARCHIVE/BIN

    Include Directories - write
    PATH-TO-FOLDER-WITH-EXTRACTED-ARCHIVE/INCLUDE

    Library Directories write
    PATH-TO-FOLDER-WITH-EXTRACTED-ARCHIVE/LIB

    Where PATH-TO-FOLDER-WITH-EXTRACTED-ARCHIVE can be C:\Program Files\GDK4\.

And it's end. Now you can start writing your own gamemodes!
